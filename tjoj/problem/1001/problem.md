# [1000: A+B Problem !!!](https://acm.tongji.edu.cn/problem.php?id=1000)

时间限制 : 1.000 sec  内存限制 : 256 MB

## 题目描述

Calculate a+b

## 输入

Two integer a,b (0<=a,b<=1000)

## 输出

Output a+b

## 样例输入

```
1 2
```

## 样例输出

```
3
```

## 提示

Just do it!

## 备注

该题目的判断数据似乎有误。当定义 `e[11][11]` 时，无法通过；而定义成 `e[21][21]` 就可以通过。而题目说点的个数不超过 10 个，那么 `e[11][11]` 应当是合理的。
